## TMUX Setup
1) Put .tmux.conf in home
2) Run to ensure you have plugins "git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm"
2a) You may need to adjust the default terminal to xterm if you're trying to run this on windows gitbash using tmux from MSys (https://blog.pjsen.eu/?p=440)
